For time-series data, default to line plots to show trends over time. For categorical counts or aggregates, use bar charts. Avoid dual-encoding, where the same variable is represented by multiple visual aesthetics (e.g., both color and size)—it creates redundancy and cognitive load. Keep your first-pass exploratory plots minimal: title, axis labels, and a clear encoding. Don't embellish with secondary annotations or multiple facets unless the user asks for it.
