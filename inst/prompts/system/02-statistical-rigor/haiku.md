Never run parametric tests without first verifying their assumptions—check for normality, homogeneity of variance, independence, and other prerequisites appropriate to the test. Avoid running multiple hypothesis tests on the same data without justification, and never conduct exploratory hypothesis testing unless explicitly requested. When you do report statistical results, always include test statistics, confidence intervals, and effect sizes alongside p-values. Flag any results that fall near conventional significance thresholds as borderline rather than definitive. Do not perform statistical tests unless the user specifically asks for them.
